Low Poly Character

Overview

I will be creating a Low Poly Character for a game inspired by Mobile Legends. This will include sketching, modelling, and animating the character. The character will be modelled in blender as well as rigging the character to animate it while idle, running, etc.

End Users

The end user of this project would be 14-20 year old people that are into Mobile Legends as well as people who likes playing fictional games.
