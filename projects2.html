Project 2: Python Coded Game

Design Considerations

Overview

I developed a Python-based word game inspired by Hangman. Each round features scrambled words, and players can replay the game multiple times for new word challenges.

End Users

The target audience for this game includes individuals aged 6 and above who enjoy word puzzles, English, or vocabulary challenges. Users should have an easy, error-free experience and the option to choose difficulty levels for added excitement.

Requirements
  • A list of words
  • A randomizer to shuffle words
  • Multiple difficulty levels
Specifications
  • The game should display and scramble words from a list.
  • It must handle different inputs without causing errors.
  • It should be accessible and easy for anyone to play.

Relevant Implications

Aesthetics

I ensured the code was clean and readable through consistent indentation and spacing. Clear text formatting helps future edits and improves overall code clarity.

Usability

I tested the program multiple times and gathered peer feedback to ensure that players experience no crashes or logic errors during gameplay.

Functionality

To meet user expectations, I ensured smooth performance and balanced difficulty levels. The program operates as intended while maintaining a fun and engaging experience.
